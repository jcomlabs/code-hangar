# Fixtures

Fixtures are required before real scanning is connected.

## Fixture set A: Markdown project

Contains:

- README.md
- AGENTS.md
- docs/overview.md
- prompts/system.md
- src/main.ts
- package.json

Purpose:

- UI navigation
- Markdown preview
- context priority
- document_fts

## Fixture set B: Sensitive files

Contains:

- README.md
- .env
- credentials.json
- token.json

Purpose:

- preview blocked
- FTS exclusion
- Protected Zone behaviour

## Fixture set C: Git-like project

Contains:

- .git/config
- README.md
- AGENTS.md
- src/main.rs

Purpose:

- local Git reader
- origin URL as passive metadata
